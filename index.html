Programa de fidelidade com clusterização de clientes

Para a realização desse projeto utilizei Python, estatística e técnicas não-supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de performance de compras.
Com o objetivo de selecionar grupos de clientes para formar um programa de fidelidade, o objetivo é aumentar a receita da empresa.
